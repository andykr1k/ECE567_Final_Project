## 🤝 How to Contribute

To keep the project organized and collaborative, please follow this workflow:

1. **Create or assign an issue**
   - Use GitHub Issues to describe bugs, tasks, or features
   - Assign yourself to the issue
   - Use the **Kanban board in Projects** for issue assignment and tracking

2. **Create a branch**
   - Always branch off `main`
   - Use a descriptive name (e.g., `feature/pose-tracking`, `fix/projection-bug`)

3. **Make your changes**
   - Keep commits clean and descriptive

4. **Submit a pull request**
   - Link the **corresponding issue**
   - Include a **brief description** explaining:
     - How the bug was fixed or the feature was implemented

5. **Review & merge**
   - Ensure your PR is reviewed before merging
